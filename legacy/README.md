# Legacy 文件夹说明

## 定位与作用

本文件夹用于存放项目**历史文档**和**参考资料**。

### 为什么需要 legacy 文件夹？

1. **保留历史记录**：某些文档虽然不再使用，但可能对项目发展历史有参考价值
2. **保持主目录整洁**：让项目主目录只包含当前活跃使用的文件
3. **避免误删**：将不用的文件集中存放，而不是直接删除

### 当前内容

- `功能清单与改进方向.md` - 旧版功能清单（v0.7.0 之前的规划）

### 文件管理原则

- 当前项目运行时**不需要**本文件夹中的任何文件
- 开发新功能时，主目录的文档和代码具有最高优先级
- 本文件夹仅作为归档参考
