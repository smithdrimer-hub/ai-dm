Text Files:  

At times, you'll need to make a last minute change - gender is a common reason for this.  We've
included a simple TXT copy of each game along with the Acrobat File, so that you can import it
into your Word Processor, and print out a copy.   

Please note: do not attempt to generate the game from the ASCII files.  The ASCII files may not
contain important graphics or other information, and are intended only to include those portions of
the game that might need to be reprinted because of changes in casting, venue, etc.
